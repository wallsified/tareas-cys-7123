## Dependencias

### Python

Se usa [uv](https://docs.astral.sh/uv/) como manejador de paquetes, entorno virtual y de ejecución. Para sincronizar la configuración se hace lo siguiente:

```bash
uv sync
```

Posteriormente, para ejecutar el programa se hace lo siguiente:

```bash
uv run numeroDelEjercicio.py
```
